First, we need to do multiplication. We can write \(3z^2 \times 2z^3\) as \(3 \times z^2 \times 2 \times z^3\text{,}\) which we can rearrange as \(3 \times 2 \times z^2 \times z^3\text{.}\) First do \(3\times 2\text{,}\) which is 6. Add the exponents: \(z^2 \times z^3 = z^5\text{.}\) So, we have \(6z^5 + z^5\text{.}\) Now we must add like terms: \(6z^5\) and \(z^5\) are like terms, so adding those gets \(7z^5\text{.}\) Thus, our answer is \(\boxed{7z^5}\text{.}\)
