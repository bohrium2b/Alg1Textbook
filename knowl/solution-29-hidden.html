Well, let’s start by expanding \(x^3\text{,}\) which we write as \(x \times x \times x\text{.}\) We will now expand \(x^2\text{,}\) which we write as \(x \times x\text{.}\) Let’s write this out fully: \({ \color{blue}x \times x \times x} \times {\color{red} x \times x}\text{.}\) Now, we can write that as an exponent, similar to what we did in Lesson 1. We are multiplying x by itself 5 times, so we get \(x^5\) as our final answer.
