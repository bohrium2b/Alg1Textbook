Solution 2.2.5.1.

Let’s do this 2 ways. First, we can do it the long way that writes it out, and then let’s do it the short way. First up, let’s do the long way! We can write \(x^4\) as \(x \times x \times x \times x\) and \(x^2\) as \(x \times x\text{.}\) Multiplying those together, we get \(x \times x \times x \times x \times x \times x\) or \(x^6\text{.}\) Let’s do it the 2nd way. We will add the exponents first: 4 \(+\) 2 is 8, so x\(^4\) \(\times\) x\(^2\) is x\(^6\text{.}\)
