What’s wrong with this solution?

Bogus Solution.

There isn’t any exponent for y, so the exponent for y stays as 3. Thus, the solution is \(y^3\text{.}\)
Remember, if no exponent is written, you should write the exponent as 1. Thus, the expression would be \(y^3 \times y^1\text{,}\) and we can simplify that to be \(y^4\text{.}\)
